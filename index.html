  • How to use Grid-template-Areas property to dictate not just how many rows and columns a grid has, but which cells occupy more than 1 column or row.   • How to use Grid-Area property to dictate this same info on-the-fly for individual items, without pre-determining the whole grid layout.
  • To use the repeat function within the grid-template-rows OR -columns property to save time when specifying multiple columns/rows of the same size. IE 100px 100px 100px becomes repeat(3, 100px).

  • To use auto-fill to automatically add and subtract columns based on the media size.
  • auto-fit to automatically adjust the size of rows/columns as the media size changes.

2020.08.14-15 - CSS Grid

Control how many columns a grid item in a CSS Grid will consume using: grid-column, which is a CSS property that applies to the grid items and not the grid itself.

grid-column: 1 / 3; makes an item stretch from the first to the third invisible line, separating all of the columns. These lines count from 1 at the top and left, upward in integers.

Same goes for rows via grid-row: 1/3;

Control the left-right justification of content within a single cell via: justify-self. This defaults to stretch but will also take arguments such as: start (aligning content to left), center, and end, whichi aligns content right.

Control the up-down alignment of a single cell in the same way as above, with all the same arguments, by using align-self.

Control the left-right justification of all cells via justify-items and presumably the up-down alignment via align-items;.

Diagram and set which cells are to be merged in a CSS Grid by using grid-template-areas. Do this by:


grid-template-areas:
"header header header"
". content content"
"footer footer footer"

2020.08.13 - Gap in between Rows via CSS Grid

Use the CSS Argument: grid-row-gap: 5px;.

The number can be relative or absolute. It will create a gap in between the horizontal rows in the CSS Grid.

You can also use grid-gap: 5px 10px; to set both the row and the column gap at once. Row gap is set w first number, column gap with the second. One number only would set BOTH row and column gap.
